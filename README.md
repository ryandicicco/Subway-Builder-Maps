# Subway-Builder - Amsterdam (AMS)
<p>Made using Kronifer's Subway Builder map patcher and CBS (Statistics Netherlands) neighborhood data for population and jobs.</p>
## Coverage:
<p>Amsterdam (all neighborhoods), Amsterdam-Noord, Zuidoost/Bijlmer, Zuidas, Westpoort, Zaandam, Purmerend, Diemen, Amstelveen, Haarlemmermeer. Schiphol (EHAM) with all piers. 31 universities including UvA and VU Amsterdam. Haarlem, Hilversum, and most of Almere are just outside the bbox. </p>
