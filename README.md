## How to install

You'll need [Railyard](https://subwaybuildermodded.com/railyard/), the community map installer for Subway Builder. Once it's running:

1. Open Railyard and head to the **Browse** tab.
2. Find the map you'd like to install by typing in the search bar or applying any of the filters on the left.
3. Click on the map, then click the download icon in the top right.
4. Next time you click **Launch**, Railyard will have already imported the map for you and it will appear right beside your other maps that you've installed. 
That's it. No manual file moving, no editing game files.

## Updates

Every map has a manifest at `releases/{CODE}.json` that Railyard checks for new versions. When I ship an update, Railyard notices, flags the new version in the app, and one click pulls it down. Full version history for each map lives in its JSON file if you want to see what changed and when.
